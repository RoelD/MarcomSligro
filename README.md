# Sligro Lokaal Limburg – v4

Deze versie is opnieuw uitgewerkt met extra aandacht voor consistente layout, nette subpagina's en een beter Sligro-geïnspireerd logo.

## Inhoud

- `index.html` – campagnehub / homepage
- `assortiment.html` – productoverzicht met filters
- `leveranciers.html` – leveranciersverhalen en contentpakket
- `events.html` – proeverijen, sampleboxen en eventprogramma
- `campagne.html` – Touch–Tell–Sell-campagneplanning
- `contact.html` – interessepagina
- `404.html` – foutpagina
- `assets/photos/` – ultrarealistische fotografie in JPG en WebP
- `assets/logo-sligro.svg` – Sligro-geïnspireerd campagnelogo
- `styles.css` – volledige styling
- `script.js` – menu, filters en micro-interacties

## GitHub Pages

1. Pak de zip uit.
2. Upload de inhoud van de map naar de root van je repository.
3. Zet GitHub Pages aan via **Settings > Pages**.
4. Kies branch `main` en map `/root`.
5. Publiceer.

De site is volledig statisch en heeft geen build-stap nodig.
