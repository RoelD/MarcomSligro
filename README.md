# Sligro Lokaal Limburg — statische GitHub Page

Deze map bevat een complete statische website voor GitHub Pages.

## Bestanden
- `index.html` — alle pagina-inhoud en navigatie
- `styles.css` — styling in Sligro-geïnspireerde look & feel
- `script.js` — mobiele navigatie, actieve navigatie en productfilters
- `assets/` — vectorafbeeldingen als SVG

## Uploaden naar GitHub Pages
1. Upload alle bestanden naar je GitHub repository.
2. Ga naar **Settings > Pages**.
3. Kies bij **Build and deployment** de branch, meestal `main`.
4. Kies de map `/root` als je alles in de hoofdmap zet.
5. Sla op en wacht tot GitHub Pages de site publiceert.

## Aanpassen
- Producten wijzig je in `index.html` bij de sectie `#assortiment`.
- Kleuren wijzig je in `styles.css` onder `:root`.
- Afbeeldingen staan als bewerkbare SVG-bestanden in `assets/`.
